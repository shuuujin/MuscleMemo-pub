# プライバシーポリシー

**制定日：2025年12月11日**

## 1. 収集する情報
本アプリ「MuscleMemo」（以下、本アプリ）は、ユーザーの個人情報を収集・送信しません。ユーザーが入力したトレーニング記録、種目データ、設定情報などは、すべてユーザー自身のデバイス内のストレージにのみ保存されます。

## 2. データの利用目的
本アプリ内で保存されたデータは、以下の目的でのみ利用されます。
* アプリ内でのトレーニング履歴の表示
* 統計情報やグラフの生成
* 1RM（最大挙上重量）の推定計算

## 3. 第三者への提供
本アプリは、ユーザーのデータを第三者のサーバーへ送信したり、第三者と共有したりすることはありません。また、広告配信SDKや解析ツール（Google Analyticsなど）による個人情報の収集も行っていません。

## 4. データの削除
本アプリをデバイスから削除（アンインストール）することで、保存されたすべてのデータは完全に削除されます。開発者がユーザーのデータを復元することはできません。
